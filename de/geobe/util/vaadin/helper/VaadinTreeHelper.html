        • void buildTree(java.util.Collection<T> roots, groovy.lang.Closure<java.util.Collection<T>> kidCollector)

          build a vaadin tree for a given tree data structure. To ease the work with tree selections, id's for the vaadin tree node consist of Tuple2 objects with a key describing the kind of object represented by the node and a value, typically a database key or other unique value used to lookup the domain instance represented by this tree node

          Parameters:
          roots - an array of root nodes
          kidCollector - closure that extracts a collection of child nodes

        • java.lang.Object findMatchingId(java.util.Map match)

          find matching itemId in the tree by comparing each id with the match parameter. This is necessary because Tree uses Identity and not Equality to compare ids. So you cannot directly look for an equal but not identical key

          Parameters:
          match - a map that looks like the id we are looking for
          Returns:
          the found tree id or null, if none
